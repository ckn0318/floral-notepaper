# 便签使用说明

便签窗口大小可以缩放，建议拖动小窗口将其甩到屏幕最右侧，放大至半屏阅读说明

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/b9ccb260-5e3e-4b01-b5d6-86253a46fbcd.png "width=184")

缩放：按住ctrl+鼠标滚轮，可以缩放窗口内的文本和图片显示大小

便签支持基本markdown语法和粘贴图片功能

<br />

# 1.便签

## 1.1 快捷键-ctrl+space

_ctrl+space是本便签的常用快捷键_

（1）便签在托盘时，按下`ctrl+space`呼出编辑窗口

（2）当便签已在桌面显示时，如果你正在使用其他软件（如微信聊天窗口），按下`ctrl+space`会切回便签，并把光标定位到编辑区，方便你直接输入内容。

## 1.2 编辑与打开

便签内容改动时，编辑器默认1s后自动保存（左下角显示字数与保存状态）

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/c9f05d65-93af-424b-83ac-73cc5b84a433.png)

保存在历史便签中，并记录最后修改时间

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/6a4f40bc-fe55-4011-a7ba-56e670c7b4e5.png "width=210")

历史便签保存在安装目录下的notes文件夹内，可以用其它md编辑器打开（图片保存在image中，相对路径）

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/d87ba8ae-e2d8-468b-bfdc-a91d6e48c4b2.png "width=338")

新建空白便签-右下角

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/59855a44-bb97-4fd5-9fee-7dfd8cd1d5fa.png)

# 2.待办清单-类灵动岛

通过右上角第一个图标可以呼出待办清单窗口

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/90851bb4-b7bf-417c-94d1-81118d1ebe16.png)

待办清单在屏幕顶部边缘时，鼠标离开窗口会收缩，变成类似灵动岛的状态栏

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/91797b38-4708-4277-a6aa-6ad7ed0ffddc.png)

右键待办条目可以设置事件优先级或置顶事件

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/7461da66-daf1-403a-b371-d758b7f36167.png "width=210")

小状态栏会根据未完成待办事项变化，显示事件优先级和待办事件数目

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/ce6370aa-9e58-4ef3-a44f-2e77c384ff80.png)

<br />

# 3.磁贴

右上角第二个图标为转为磁贴模式，磁贴模式下不可编辑，只可以缩放显示大小。

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/d9376a99-c7d9-47c6-a9fb-8c4f034e6353.png "width=272")

快捷键——在磁贴模式下，右键窗口可以选择复制或转回小窗，也可以按下`ctrl+space`转回小窗

![](images/8dfb8f81-1497-4b7c-9b8a-747cab8adc2d/0b2ff98b-abd1-47e6-80e5-c547b0471e11.png "width=267")
