# 便签使用说明

便签窗口大小可以缩放，建议拖动小窗口将其甩到屏幕最右侧，放大至半屏阅读说明

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/b9ccb260-5e3e-4b01-b5d6-86253a46fbcd.png "width=184")

缩放：按住ctrl+鼠标滚轮，可以缩放窗口内的文本和图片显示大小

便签支持基本markdown语法和粘贴图片功能

<br />

# 1.便签

## 1.1 ctrl+space是本便签的常用快捷键

（1）便签在托盘时，按下`ctrl+space`呼出编辑窗口

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/5c5e9194-1f48-425d-b2c9-f0dd561b993d.png "width=209")

右上角图标为转为磁贴模式，磁贴模式下不可编辑，只可以缩放显示大小。

（2）当便签已在桌面显示时，如果你正在使用其他软件（如微信聊天窗口），按下`ctrl+space`会切回便签，并把光标定位到编辑区，方便你直接输入内容。

（3）在磁贴模式下，按下`ctrl+space`可以转回编辑器，也可以按右键-转为小窗

## 1.2 编辑的便签内容会自动保存在历史便签中，并记录最后修改时间

内容改动时，编辑器默认1s后自动保存

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/4dcbe5e1-b33d-4976-8b08-baa3662fb635.png "width=547")

## 历史便签（md文件）保存在软件安装目录下的note文件夹内，可以用其它md编辑器打开

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/d87ba8ae-e2d8-468b-bfdc-a91d6e48c4b2.png "width=557")

## 1.3 右下角-新建空白便签

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/3fd1d136-262e-47fc-94b1-d83f7531a32f.png "width=210")

# 2.待办清单

通过右上角图标可以呼出待办清单窗口

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/90851bb4-b7bf-417c-94d1-81118d1ebe16.png)

待办清单在屏幕顶部边缘时，鼠标离开窗口会收缩，变成类似灵动岛的状态栏

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/91797b38-4708-4277-a6aa-6ad7ed0ffddc.png)

右键待办条目可以设置事件优先级或置顶事件

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/7461da66-daf1-403a-b371-d758b7f36167.png "width=210")

小状态栏会根据未完成待办事项变化，显示事件优先级和待办事件数目

![](images/2fd9e082-c6cd-4766-94c5-5fa4dac337b3/ce6370aa-9e58-4ef3-a44f-2e77c384ff80.png)
