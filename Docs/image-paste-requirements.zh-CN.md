# 粘贴截图图片功能需求对齐

## 背景

当前目标是在便签编辑器中支持从剪贴板粘贴截图图片。

目标链路：

1. 用户截图后，图片进入系统剪贴板。
2. 用户在便签正文编辑区按 `Ctrl+V`。
3. 应用保存图片文件，并在正文光标处插入 Markdown 图片引用。
4. 预览区、重启后的笔记、磁贴或小窗中能正常显示图片。

## 预期验收标准

请确认或修改：

- [x] 在正文编辑区按 `Ctrl+V` 粘贴截图后，正文自动插入 Markdown 图片引用。
- [x] 切换到预览或分栏模式，图片能正常显示。
- [x] 保存并重启应用后，图片仍能显示。
- [x] 删除笔记时，该笔记关联图片按确认后的规则处理。
- [x] 粘贴非图片内容时，原有文本粘贴行为不受影响。
- [x] 图片过大或格式不支持时，显示可理解的错误提示。

## 当前建议的最小实现范围

建议第一版 PR 聚焦：

1. 支持 `Ctrl+V` 粘贴系统剪贴板截图。
2. 图片保存到应用数据目录下的 `images/<noteId>/`。
3. 正文插入 `![](images/<noteId>/<file>.png)`，预览区解析并显示。

待确认后再决定是否加入文件选择、导出附件、自动压缩等扩展能力。
